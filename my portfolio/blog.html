Projects By Aniket!

Student Amenity Center

This is a RDBMS project developed using HTML,CSS,Javascript(front-end) and Php,MySQL (backend). This project... will enable the user to buy and sell his amenities, The user shall login to the system and check for availability of items. The admin shall check for the availability of the items to the customer. The user can make payment only in cash on delivery. All the data regarding the products are stored in MySQL database. The user has to enter his name, address, phone details .The UI is attractive and the connectivity to back end is robust. The main advantage is that the Students can buy college stationary items in weekends also.

Text Analyzer app

This application is build using ReactJs,HTML and CSS to analyze a text typed in the text box.... It will calculate the no.of words,letters,vowels and the time to read the text.Buttons like Uppercase ,lowercase,clear are added which will convert the text to uppercase, to lowercase and clear the text.
